
# myUtilsApp

[English Document](https://github.com/LiuYueKai/myUtilsApp)

## 介绍
解决Typora不支持目录的唯一痛点，打造完美MD编辑器。

## 功能
* 支持选择根目录，根据根目录显示目录结构
* 点击目录节点后通过Typora打开对应文档

### 安装
```

$ cd myUtilsApp
$ npm i
```
